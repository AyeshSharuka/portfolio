## Overview

Personal portfolio website showcasing my experience, projects, and technical skills as an AI Engineer. Built with a clean, professional design focused on readability and performance.

## Tech Stack

- **React 18** + **TypeScript**
- **Framer Motion** - page and scroll animations
- **Lucide React** - icon library
- **CSS Custom Properties** - design token system
- **Vercel** - deployment and hosting

## Features

- Typewriter cycling title animation
- Smooth scroll animations on all sections
- Responsive design - mobile, tablet, desktop
- Sticky navigation with mobile menu
- Project cards with featured highlights
- Work experience timeline
- Skills grouped by category
- Professional recommendation section
- One-click CV download

## Project Structure

```
src/
├── components/
│   ├── Navbar.tsx        # Sticky nav with mobile hamburger menu
│   ├── Hero.tsx          # Hero with typewriter animation + photo
│   ├── About.tsx         # About section with pillar cards
│   ├── Experience.tsx    # Work history timeline
│   ├── Projects.tsx      # Project cards grid with featured badges
│   ├── Skills.tsx        # Skills grouped by category
│   ├── Education.tsx     # Education + LinkedIn recommendation
│   └── Contact.tsx       # Contact info + footer
├── data/
│   └── portfolio.ts      # Centralised content file
├── index.css             # Global styles and design tokens
└── App.tsx               # Root component
```

## Running Locally

```bash
npm install
npm start
```
